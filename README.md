A lightweight static website for accepting donations via YooMoney and alternative payment methods.

This project provides a clean, minimal, and responsive donation page with a configurable payment amount and a dedicated section for bank transfer details. The primary payment form redirects users to YooMoney to complete the transaction, which means that bank card details are never entered, stored, or processed by the website itself.

The site is designed to be simple to deploy, easy to customize, and suitable for personal projects, communities, open-source initiatives, or any other cases where a straightforward donation page is needed.
